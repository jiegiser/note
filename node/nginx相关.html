    worker_connections  1024;

# nginx对应的操作命令

测试配置文件格式是否正确 nginx -t 启动 nginx 重启：nginx -s reload 停止: nginx -s stop

注意在Windows环境下，需要使用管理员权限打开cmd然后进行操作。
