# PostGIS 安装

# 软件下载

postgresql

https://www.postgresql.org/download/windows/

选择 9.6.19 的版本；因为 postgis 只支持 9.6.12 的版本；

postgis-bundle-pg12-3.0.2x64.zip

http://download.osgeo.org/postgis/windows/pg12/

# 将 postgresql.zip 解压

不要放在含有中文名的文件夹中，直接右击解压到当前文件夹即可，例如我解压到了 D:\pgsql 文件夹中；

# 创建数据存放目录

在解压后的文件夹中新建 data 文件夹；（D:\pgsql\data）

# 初始化数据库

使用管理员身份打开 cmd，运行命令： D:\pgsql\bin\initdb.exe -D D:\pgsql\data -E UTF8 --locale=Chinese

# 启动数据库，有两种方式

# 以注册服务方式

  1. 注册服务

D:\pgsql\bin\pg_ctl.exe register -D D:\pgsql\data -Npgsql

-N 表示 windows 服务名称为 pgsql；

  1. 启动服务

net start pgsql

如果你的安装没有错误，现在就应该可以起来了。

下面是一些服务的其他操作：

  1. 关闭服务

net stop pgsql

  1. 卸载服务

D:\pgsql\bin\pg_ctl.exe unregister -D D:\pgsql\data -Npgsql

# 直接启动方式

  1. 启动

D:\pgsql\bin\pg_ctl.exe start -w -D D:\pgsql\data

  1. 关闭

D:\pgsql\bin\pg_ctl.exe stop -W -D D:\pgsql\data

# 创建数据库

D:\pgsql\bin\createdb.exe -E UTF8 geodb

D:\pgsql\bin\dropdb.exe geodb

# 创建用户

# 使用 
createuser [选项]...[用户名]
# 如下
# 会有是否创建 superuser 的选项，创建一个名为 postgres 的超级用户；
D:\pgsql\bin\createuser.exe -s -r postgres

选项: -c, --connection-limit=N 角色的连接限制(缺省: 没有限制)

-d, --createdb 此角色可以创建新数据库

-D, --no-createdb 此角色不可以创建新数据库

-e, --echo 显示发送到服务端的命令

-E, --encrypted 口令加密存储

-i, --inherit 角色能够继承它所属角色的权限

（这是缺省情况)

-I, --no-inherit 角色不继承权限

-l, --login 角色能够登录(这是缺省情况)

-L, --no-login 角色不能登录

-N, --unencrypted 口令不加密存储

-P, --pwprompt 给新角色指定口令

-r, --createrole 这个角色可以创建新的角色

-R, --no-createrole 这个角色没有创建其它角色的权限

-s, --superuser 角色将是超级用户

-S, --no-superuser 角色不能是超级用户

--help 显示此帮助信息, 然后退出

--version 输出版本信息, 然后退出

联接选项:

-h, --host=HOSTNAM 数据库服务器所在机器的主机名或套接字目录

-p, --port=PORT 数据库服务器端口号

-U, --username=USERNAME 联接用户 (不是要创建的用户名)

-w, -no-password 永远不提示输入口令

-W, --password 强制提示输入口令

查看系统中的所用用户：select * from pg_user 删除一个用户：drop user dan;其中 dan 为用户名 或者：D:\pgsql\bin\dropuser.exe dan

# postgis 安装

解压 postgis-bundle-pg12-3.0.2x64.zip 到没有中文或者空格的目录。

然后修改 makepostgisdb_using_extensions.bat 文件，配置数据库以及 pg 数据库：

set PGPORT=5432
set PGHOST=localhost
set PGUSER=postgres
set PGPASSWORD=gis
set THEDB=geodb
set PGINSTALL=D:\pgsql

然后双击执行这个 makepostgisdb_using_extensions.bat 文件即可；

# 将空间数据导入 PostGIS 中
