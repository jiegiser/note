# URI、URL、URN 是什么以及他们之间的区别

# URI

  • URI （统一资源标志符）就是包含 URL 和 URN 他们统一的定义；
  • 用唯一标识互联网上的信息资源；

# URL

  • Uniform Resource Locator 统一资源定位器；根据地址进行访问资源。
  • 格式：http://user:pass@host.com:90/path?query=string#hash;协议：http://（不同的协议解析方式是不同的）
  • 此类格式的都叫做 URL，比如 ftp 协议

# URN

  • 永久统一资源定位符
  • 在资源移动之后还能被找到
  • 目前还没有非常成熟的使用方案
