# linux 简单操作

  • 登录：ssh work@192.168.2.123 回车输入密码登录
  • 查看文件夹：ls：查看文件平铺；ls -a: 查看所有文件，包括隐藏文件；ll：查看文件列表；
  • 创建文件夹：mkdir 文件夹名；查看文件夹中内容：ll 文件夹名；
  • 删除文件夹：rm -rf 文件夹名；
  • 修改文件名：mv 要修改的文件名 修改后的文件名；eg: mv index.html index1.html
  • 移动文件：mv 要移动的文件名 移动到的目录的地址；eg：mv bundle.js ../bundle.js(可以按 tab 键进行补全文件名)
  • 拷贝文件：cp 要拷贝的文件名 拷贝后的文件名；eg：cp a.js a1.js:将 a.js 拷贝为 a1.js；
  • 删除文件：rm 文件名；eg：rm a.js：删除 a.js 文件
  • 新建文件：1. touch 文件名；eg: touch d.js; 2. vi 文件名；eg: vi d.js：新建一个 d.js 文件，随即打开 vim 编辑器可以进行编辑，点击 i 进行插入。点击 esc 退出编辑，然后输入:w 保存，然后输入 q ，退出编辑；
  • 查看文件内容：1. 使用 vi 文件名，或者 vim 文件名；eg：vi d.js ,如果要打开的文件没有，就会进入编辑模式；2. cat 文件名：打印文件中所有内容；3. head 文件名：只打印文件的前面部分，4. tail 文件名：打印文件最后几行内容；
  • 查找文件中的内容：grep 需要查找的内容 要查找的文件名；eg：grep "babel" package.json
  • vim 编辑器：i 键是插入内容，通过上下左右进行切换插入的位置。:q! 为强制退出。
